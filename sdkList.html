Third-party SDK List

Third-Party SDK List

Update Date: 2026-01-01

To provide a better service, Mooban uses the following third-party SDKs in the following ways:
🌐
Alamofire
Swift HTTP Network Library · v5.11.1
Provider Alamofire Software Foundation
Usage Purpose Handles HTTP requests between App and server over the HTTP/HTTPS protocol
Data Collection Policy No user data is collected by this SDK. It is used as a network communication tool
License MIT License
Project URL github.com/Alamofire/Alamofire
🖼️
Kingfisher
Swift Image Loading and Caching Library · v8.8.1
Provider Wei Wang (onevcat)
Usage Purpose Loads and caches network images, including user avatars, dynamic images, topic covers, etc., to improve image loading performance
Data Collection Policy No user data is collected by this SDK. Image cache files are stored in the local sandbox and can be deleted by clearing the cache
License MIT License
Project URL github.com/onevcat/Kingfisher
💬
MijickPopups
SwiftUI Popup / Toast Component Library · v4.0.5
Provider Mijick
Usage Purpose Provides App toast notifications, bottom popups, operation menus, etc.
Data Collection Policy No user data is collected by this SDK.
License MIT License
Project URL github.com/Mijick/Popups
